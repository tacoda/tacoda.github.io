# caml

<a href="{{ github_link }}"><i class="fab fa-github">&nbsp; <i class="fas">{{ title }}</i></i></a>

`caml` (pronounced *camel*) is a dynamic CLI tool defined with declarative YAML.

Simply create a `caml.yaml` file and `caml` will build a CLI tool from it.
